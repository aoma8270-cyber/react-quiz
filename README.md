# ⚛️ React Quiz App

Reactの基礎概念（コンポーネント設計、Hooksによる状態管理）の理解と、モダンなフロントエンド開発フロー（Vite + Vercel）を習得するために作成したクイズアプリケーションです。

## 🌐 デモ (Live Demo)
**https://react-quiz-livid-six.vercel.app/**
*(PC・スマートフォン等のレスポンシブ対応済み)*

## 🛠️ 使用技術 (Tech Stack)

| Category | Tech |
| --- | --- |
| **Frontend** | React, CSS |
| **Build Tool** | Vite |
| **Deploy** | Vercel |
| **Version Control** | Git / GitHub |

## 💡 実装機能 (Features)

* **クイズ出題機能:** 配列データで管理された問題セットを順次表示。
* **インタラクティブな判定:** 回答クリックと同時に正誤判定を行い、UIに即座に反映。
* **スコア管理:** `useState` を活用し、正解数と進行状況をリアルタイムに管理。
* **リザルト画面:** 全問終了後にスコアとフィードバックを表示。
